# DapoerMuda POS

Aplikasi kasir mobile berbasis HTML, CSS, dan JavaScript.

## Fitur

- Login admin dan kasir
- Transaksi penjualan
- Manajemen stok dan restok
- Riwayat transaksi
- Rekap pemasukan mingguan dan bulanan
- PWA installable untuk Android
- Wrapper Android native dengan Capacitor

## Menjalankan

Jalankan lewat web server lokal atau hosting statis. Jangan buka hanya dengan klik file `index.html` jika ingin fitur instal aplikasi aktif.

## Instal di HP

1. Upload project ke hosting HTTPS.
2. Buka link aplikasi di Chrome Android.
3. Pilih install atau tambahkan ke layar utama.

## Build APK Android

Project ini sekarang juga sudah punya folder `android/` untuk build APK native.

APK debug hasil build ada di:

`android/app/build/outputs/apk/debug/app-debug.apk`
